HTML Semântico:

O HTML semântico tem como objetivo descrever o significado do conteúdo presente em documentos HTML, tornando-o mais legível e claro, tanto para programadores quanto para browsers e, respectivamente, leitores de tela.

<header>

Utilizado para representar o cabeçalho de um documento ou seção declarado no HTML
<section>

Representa uma seção dentro de um documento e geralmente contém um título
<article>

Utilizamos esse elemento quando precisamos declarar um conteúdo que não precisa de outro para fazer sentido em um documento HTML. Ex: um artigo em um blog

<nav>

O elemento <nav> é utilizado quando precisamos representar um agrupamento de links de navegação, que, por sua vez, são criados com os elementos <ul>, <li> e <a>.</p>

<aside>

O elemento <aside> é utilizado quando precisamos criar um conteúdo de apoio/adicional ao conteúdo principal. Ex: sugestões de leituras complementares em sites

<main>

Esse elemento especifica o conteúdo principal e de maior relevância dentro da página. Para ser considerada bem construída, uma página deve apresentar apenas um conteúdo principal.
<figure>

Esse elemento é usado para a inserção de uma figura. Para incluir a descrição podemos utilizar <figcaption>.
<footer>

Esse elemento representa um rodapé, como a área presente no final de uma página web.
